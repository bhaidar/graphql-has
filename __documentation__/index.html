Setup

npm install

First run after the setup

If you are on production you just need to run the npm run prod command.

After having changed the schema or GraphQL definitions

This application is capable to automatically create fragments, typescript interfaces and the schema.graphql file for you... it also intercepts changes on graphql definitions and, if any, triggers the generation process during the bootstrap. If you want to execute this procedure manually just run npm run prepare-graphql

About autogenerated interfaces

It is important, after having added or changed something to the GraphQL definitions, to generate the interfaces before using them to implements our custom classes. To do so, you can run/restart the server immediately after or use the wizard (npm help).

The application will expose two endpoints to download types and interfaces ('{rest_endpoint}/types' and '{rest_endpoint}/interfaces'). You can also share these interfaces + fragments with UI WebApps.
